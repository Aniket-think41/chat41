# chat41
Question-answer prompt by using the dockerization process with groq api with llama model 
Back-End:
1. npm i 
2. npm install requirements.txt
3. export GROQ_API_KEY=use your own key  // use your own key
4. node server.js

Front-End: 
1. npm i
2. npm install requirements.txt
3. npm start
